What is It?

This is a tiny extension of jQuery UI Sortable. By default sortable allows you to connect two or more sortable elements together. Unfortunately, there is no option provided to change the sytle of the element that was moved to the style of the sortable element it was moved to. This extension adds an option to easily make that happen.

Sort items from one list into another and vice versa, by passing a selector into the connectWith option. The simplest way to do this is to group all related lists with a CSS class, and then pass that class into the sortable function (i.e., connectWith: '.myclass').

Custom Functionality after Transition

Sort items from one list into another and vice versa. Call a custom function after the transition is complete. This is done by passing stylechange as a callback function.

JavaScript:


    //Custom Functionality after Transition
    $( "#sortable7" ).sortable({
        connectWith: ".connectedSortableCallback",
        styleClass:  "ui-state-default",
        styleChangeDuration: 1000,
        stylechange: function (event, ui) {
            alert('I (sortable7) have received an item');
        }
    }).disableSelection();

    $( "#sortable8" ).sortable({
        connectWith: ".connectedSortableCallback",
        styleClass:  "ui-state-highlight",
        styleChangeDuration: 1000,
        stylechange: function (event, ui) {
            alert('I (sortable8) have received an item');
        }
    }).disableSelection();

Options

Option Description Default
styleClass Class that will be added to any sortable list item that is moved to the current sortable element. If set to null the class will not be altered. null
styleChangeDuration A string representing one of the three predefined speeds ("slow", "normal", or "fast") or the number of milliseconds to run the transition animation (e.g. 1000). 500
styleChangeEasing A string indicating which easing function to use for the transition. swing

Click here for a list of the options provided by jQuery UI Sortable.
